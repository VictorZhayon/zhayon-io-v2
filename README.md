# Zhayon.io V2

Welcome to **Zhayon.io V2**, my personal portfolio and professional hub. This project showcases my recent work, skills, and professional experience, built with modern web technologies for a smooth, interactive user experience.

## Features

- **Interactive UI**: A sleek, modern design with smooth animations powered by Framer Motion.
- **Project Showcase**: Detailed project card previews to explore my latest work.
- **Command Palette**: Quick navigation and actions via a powerful command palette.
- **CV Preview**: Integrated modal to preview and download my professional CV.
- **Legacy Access**: Easy navigation back to the [V1 website](https://zhayon-io.vercel.app).
- **Responsive Design**: A fully responsive layout with a fixed sidebar for desktop and accessible mobile navigation.

## Technologies Used

This project is built with a robust, modern tech stack:

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Analytics**: Vercel Analytics

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed (recommend using nvm)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd zhayon-io-v2
   ```

3. Install the dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

This project is configured for deployment on platforms like Vercel. Push changes to the repository to trigger a production build.
